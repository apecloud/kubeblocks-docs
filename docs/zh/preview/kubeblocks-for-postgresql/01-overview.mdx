# KubeBlocks PostgreSQL 插件概述

PostgreSQL 是一款功能强大的开源关系型数据库管理系统，以其以下特点著称：
- 处理大规模数据集的可扩展性
- 强大的安全特性
- 丰富的自定义选项
- 对多样化应用的支持

KubeBlocks PostgreSQL 插件通过附加功能和扩展增强了 PostgreSQL 的能力：

## 支持的扩展

| 扩展名称      | 描述                                                                 |
|----------------|-----------------------------------------------------------------------------|
| **pgvector**   | 为AI/ML应用和相似性搜索提供向量运算能力    |

**KubeBlocks PostgreSQL 插件**为在Kubernetes环境中部署和管理PostgreSQL集群提供了完整的解决方案。主要功能包括：

- 灵活的部署拓扑结构
- 全面的生命周期管理
- 可靠的备份与恢复操作
- 支持多版本PostgreSQL

## 核心特性

### 生命周期管理

KubeBlocks通过全面的生命周期管理简化PostgreSQL运维：

| 功能特性                      | 描述                                                                 |
|------------------------------|-----------------------------------------------------------------------------|
| **水平扩展**       | 增减副本数量以调整容量                                   |
| **垂直扩展**         | 调整PostgreSQL实例的CPU/内存资源                       |
| **存储卷扩容**         | 无需停机即可动态增加存储容量                     |
| **重启操作**       | 以最小影响实现受控的集群重启                        |
| **启动/停止**               | 临时暂停/恢复集群操作                              |
| **密码管理**      | 可在创建时为PostgreSQL集群设置和管理自定义root密码                                               |
| **动态配置**    | 无需重启即可修改PostgreSQL参数                            |
| **自定义服务**          | 暴露专用的数据库端点                                      |
| **主从切换**               | 计划内的主从角色变更                                       |
| **副本管理**       | 安全地停用或重建特定副本                          |
| **版本升级**         | 无缝执行次版本升级                                  |
| **高级调度**      | 自定义Pod放置和资源分配                           |
| **TLS加密**           | 启用/禁用传输层安全协议                                   |
| **监控**               | 集成的Prometheus指标采集                                  |
| **日志**                  | 通过Loki Stack实现集中式日志管理                                           |


### 备份与恢复

KubeBlocks支持多种PostgreSQL备份策略：

| 备份类型       | 方法          | 描述                                                                 |
|-------------------|-----------------|-----------------------------------------------------------------------------|
| **全量备份**   | pg_basebackup   | PostgreSQL原生工具，用于完整数据库快照                   |
| **全量备份**   | WAL-G           | 高效的备份工具，支持压缩和云存储            |
| **持续备份**    | postgresql-pitr | 定期将PostgreSQL预写日志(WAL)文件上传至备份仓库，通常与`pg-basebackup`配合使用|
| **持续备份**    | WAL-G归档   | 定期将PostgreSQL预写日志(WAL)文件上传至备份仓库，通常与`wal-g`配合使用|

### 支持版本

KubeBlocks PostgreSQL插件支持以下PostgreSQL版本：

| 主版本 | 支持的次版本       |
|---------------|--------------------------------|
| PostgreSQL 12 | 12.14.0, 12.14.1, 12.15.0     |
| PostgreSQL 14 | 14.7.2, 14.8.0                |
| PostgreSQL 15 | 15.7.0                        |
| PostgreSQL 16 | 16.4.0                        |

可通过以下命令查看支持的版本列表：
```
kubectl get cmpv postgresql
```
