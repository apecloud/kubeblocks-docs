### Create a Cluster

You can create a ClickHouse cluster using Helm:

```bash
# Create a ClickHouse cluster with 3 shards and 2 replicas per shard
helm install ch-cluster kubeblocks/clickhouse-cluster \
  --version 1.0.1-beta.0 \
  --set shards=3 \
  --set replicas=2 \
  --set keeper.replicas=3 \
  --namespace default
```

### Configuration Options

| Parameter | Description | Default |
|-----------|-------------|---------|
| `shards` | Number of shards | 1 |
| `replicas` | Number of replicas per shard | 1 |
| `keeper.replicas` | Number of ClickHouse Keeper nodes | 1 |
| `storage.size` | Storage size per node | 10Gi |
| `resources.requests.cpu` | CPU request | 500m |
| `resources.requests.memory` | Memory request | 1Gi |
